# Pepa Martín – Model Portfolio

Portfolio web desarrollado con **React + Vite** para mostrar el trabajo de la modelo **Pepa Martín**.

Incluye galería de imágenes, vídeos y una sección about para presentar su trabajo de forma visual y minimalista.

## Tecnologías utilizadas

- React
- Vite
- JavaScript
- CSS
- HTML5

---

## Características

- Galería de imágenes optimizada
- Reproductor de vídeo integrado
- Navegación entre imágenes tipo lightbox
- Diseño minimalista para portfolios
- SEO básico optimizado
- Sitemap y robots configurados
- Preparado para producción
